# Gelateria Venice — Website

Modern, award-worthy website for **Gelateria Venice**, #1 gelato shop in Tivoli Terme.

## Features

- Full-screen hero with cinematic parallax entrance
- Animated scroll-reveal sections
- Flavors showcase grid (wide + featured cards)
- "Tre Pilastri" experience section (replaces previous 3D section)
- Ratings & testimonials from Google (4.7★), Facebook (4.9★), TripAdvisor
- Masonry gallery with lightbox
- Visit section with embedded Google Maps
- Mobile-first responsive design (375 / 768 / 1024 / 1440px)
- Accessibility: ARIA labels, keyboard nav, reduced-motion support
- Cookie consent banner
- Sticky nav with scroll-blur effect

## Design System

- **Style:** Liquid Glass · Premium luxury
- **Heading font:** Bodoni Moda
- **Body font:** Jost
- **Colors:** Warm black `#1C1917` · Gold `#D4A017` · Cream `#FAFAF9`

## Deploy

Hosted via **GitHub Pages** — push to `main` branch, enable Pages from repository settings.
